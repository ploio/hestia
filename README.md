# Hestia

[Packer][] is used to build some boxes.
See: https://atlas.hashicorp.com/portefaix

## Usage

* Install [Packer][].

* Build boxes :

        $ make buid template=xxx

* Deploy to Atlas :

        $ make push template=xxx
